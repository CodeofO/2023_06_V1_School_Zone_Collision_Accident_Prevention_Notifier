<img width="815" alt="image" src="https://github.com/CodeofO/2023_School_Zone_Collision_Accident_Prevention_Notifier/assets/99871109/c971c29d-c871-4021-99cf-7f327a27613b">


* 프로젝트 : AI를 이용한 스쿨존(골목길) 충돌사고 예방 알고리즘
* 주최 : 울산대학교 캡스톤디자인
* 제안배경 : 매년 높아지는 스쿨존 교통사고를 예방하기 위한 아이디어입니다.
* 핵심
    1. 좁은 길의 특성상 주차되어 있는 차량이 많아, 멈춰있는 객체, 움직이는 객체를 분리합니다.

    2. 움직이는 객체가 인식이 된 후 일정 조건이 되면 알림(Warning)을 제공합니다.
       <img width="883" alt="image" src="https://github.com/CodeofO/2023_School_Zone_Collision_Accident_Prevention_Notifier/assets/99871109/6baed63e-58bc-46de-b179-866c2eef4ce4">

    3. 이때 알림은 탐지된 객체에 따라서 다른 모양의 Sign을 주게 됩니다.
